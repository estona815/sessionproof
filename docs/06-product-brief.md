# Product brief

SESSIONPROOF turns the limited set of release records a creator chooses to submit into a reviewable contributor ledger. It preserves exact fragments, separates composition from master ownership, flags contradictions and missing credits, asks the smallest useful questions, records workflow acknowledgements, and exports an evidence-linked pack.

Primary job: before release, show the project owner which contributor facts are supported, conflicting, missing, or still unconfirmed.

The product does not determine rights. Legal interpretation, sample clearance, identity merging, split correction, contributor contact, export approval, deletion, refund, and pricing changes retain explicit human boundaries.
